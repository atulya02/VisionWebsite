CODE-GOLF

Shorter path enables you to achieve more with less - the challenge is figuring out how to get there. VISION, the technical society of MANIT announces CODEGOLF, which is a type of recreational computer programming competition in which participants strive to achieve the shortest possible source code for the given problem. So, grab the opportunity, scratch your brains and showcase your skills on HACKEREARTH to win exciting prizes worth ₹25,000, which includes Cash prizes, Goodies, Internship opportunities, T-Shirts, Stickers, Certificates and a lot more stuff.
